# 星空粒子生成器 v3.2

**Starry Sky Particle Generator** — Adobe After Effects 2026 ScriptUI Panel

一键在合成中生成大量动态星空粒子。支持多种形状、颜色、运动模式和特效。所有参数通过 AE 表达式驱动，生成后可通过 `Ctrl_Starfield` 控制器层实时调参。

---

## 安装

将 `starry-sky-generator.jsx` 复制到 AE 脚本目录：

```
C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\
```

重启 AE，菜单 **窗口 → 星空粒子生成器** 即可打开面板。

> 支持 AE 2026，其他版本未测试。

---

## 快速开始

1. 打开或新建一个合成
2. 调整面板参数（数量、尺寸、颜色等）
3. 点击 **▶ 生成粒子** → 弹出确认框显示参数摘要，确认后生成
4. 合成中自动创建 `Ctrl_Starfield` 控制器层，所有参数可实时微调

**核心工作流**：面板设定初始形态 → 确认参数 → 生成粒子 → 通过 `Ctrl_Starfield` 层实时调参。

## 功能详解

### 粒子参数

| 控制项 | 范围 | 说明 |
|--------|------|------|
| 数量 | 10 ~ 2000 | 粒子总数 |
| 尺寸 | 1 ~ 100 px | 最小 ~ 最大粒子尺寸 |
| 缩放 | 1 ~ 200% | 出生大小 → 消亡大小 |
| 随机缩放 | 1 ~ 200% | 每个粒子的随机尺寸变异 |
| 形状 | 方 / 圆 / 五边 / 六边 | 圆形和多边形使用 Mask Feather 模糊 |
| 颜色 | HSL | 色相/饱和度/亮度 + 色相扩散，每粒子独立颜色 |

### 发射区域

- **全合成** — 粒子均匀分布于整个画面
- **遮罩范围** — 从指定图层的遮罩内发射
- **密度百分比** — 控制遮罩内发射的粒子占比（100%=全在遮罩内，50%=一半遮罩一半全合成）

### 运动控制

| 控制项 | 范围 | 说明 |
|--------|------|------|
| 主方向 | 0 ~ 360° | 粒子主运动方向 |
| 方向扩散 | 0 ~ 360° | 每个粒子的随机偏移角度 |
| 速度 | 1 ~ 999 px/s | 最小 ~ 最大速度 |
| 目标 | 无 / Null点 / 遮罩范围 | 选择吸引目标类型 |
| 吸引力 | 0 ~ 15 (强度) | 向目标加速的力度。0=无，1=柔和，5=明显，15=瞬间 |
| 吸引时长 | 0 ~ 30 s | 粒子被吸引的持续时间 |
| 环绕 | 开/关 | 粒子到达边界后从对面折返 |

### 生命周期

| 控制项 | 说明 |
|--------|------|
| 时长 | 粒子存活时间范围（秒） |
| 淡入 | 粒子出现时的渐变过渡（秒） |
| 淡出 | 粒子消亡时的渐变过渡（秒） |

### 高级效果

| 效果 | 说明 |
|------|------|
| 闪烁 | 亮度周期性波动（可调幅度和速度） |
| 模糊 | Mask Feather 或 Gaussian Blur（可调强度和比例） |
| 随机种子 | 改变种子值可整体重置所有随机分布 |
| 发射偏移 | 粒子出现时间随机偏移（秒），制造"不齐"效果 |

### 预设系统

**快捷预设（4 个内建）**：

| 预设 | 粒子数 | 特点 |
|------|--------|------|
| 经典星空 | 300 | 蓝色系，向上飘动 |
| 彩色星云 | 500 | 全色相，360°扩散 |
| 极光飘动 | 400 | 绿色系，横向运动 |
| 金色粒子雨 | 250 | 金色，高速下落 |

**槽位预设（4 个用户槽位）** — 保存/加载当前参数，持久化存储在 AE 设置中。支持导出/导入 JSON 文件。

---

## 控制器层 `Ctrl_Starfield`

生成粒子时自动创建一个 Null 层，包含 28 个参数滑块，覆盖所有粒子行为：

- 粒子参数（数量、尺寸、缩放）
- 颜色 HSL + 色相扩散
- 运动控制（方向、速度、吸引力、吸引时长）
- 生命周期（时长、淡入淡出）
- 高级效果（闪烁、模糊、随机种子、发射偏移、发射密度）

> 所有粒子动画由 AE 表达式驱动，调整滑块后**即时生效**，无需重新生成。

## 技巧与提示

- 大量粒子（>500）建议使用圆形以优化性能
- 预设法快速定位效果方向，再细调参数
- 吸引 + 密度搭配：设置密度=50% + 吸引力=2~5，做出粒子从四周汇入遮罩范围的效果
- `Ctrl_Starfield` 层表达式可直接编辑，实现更复杂的自定义行为
- 密度变化需要重新生成方可生效（已生成粒子的表达式不会变）
