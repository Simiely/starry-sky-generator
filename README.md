# 星空粒子生成器 v3.1.2

**Starry Sky Particle Generator** — Adobe After Effects 2026 ScriptUI Panel

一键在合成中生成大量动态星空粒子，支持多种形状、颜色、运动模式和特效，所有参数通过 AE 表达式驱动，生成后可随时微调。

---

## 安装

将 `starry-sky-generator.jsx` 复制到 AE 脚本目录：

```
C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\
```

重启 AE，菜单 **窗口 → 星空粒子生成器** 即可打开面板。

> 支持 AE 2026，其他版本未测试。

---

## 快速开始

1. 打开或新建一个合成
2. 调整面板参数（数量、尺寸、颜色等）
3. 点击 **▶ 生成粒子**
4. 合成中会自动创建 `Ctrl_Starfield` 控制器层，所有参数可在其中微调

**核心工作流**：面板设定初始形态 → 生成粒子 → 通过 `Ctrl_Starfield` 层实时调参。

## 功能详解

### 粒子参数

| 控制项 | 范围 | 说明 |
|--------|------|------|
| 数量 | 10 ~ 2000 | 粒子总数 |
| 尺寸 | 1 ~ 100 px | 最小 ~ 最大粒子尺寸 |
| 初始大小 | 1 ~ 200% | 粒子出生时的缩放比例 |
| 最终大小 | 1 ~ 200% | 粒子消亡时的缩放比例 |
| 缩放随机 | 0 ~ 200% | 每个粒子缩放的随机变异范围 |
| 形状 | 方 / 圆 / 五边 / 六边 | 圆形和多边形使用 Mask，方形使用高斯模糊 |
| 颜色 | HSL | 色相/饱和度/亮度 + 色相扩散，每粒子独立颜色 |

### 发射区域

- **全合成** — 粒子均匀分布于画面
- **遮罩范围** — 从指定图层的遮罩内发射，支持密度百分比控制

### 运动控制

| 控制项 | 说明 |
|--------|------|
| 主方向 | 粒子整体运动方向（0~360°） |
| 方向扩散 | 每个粒子方向的随机偏移角度 |
| 速度 | 最小 ~ 最大速度（像素/秒） |
| 目标吸引 | 粒子被 Null 点或指定遮罩吸引 |
| 吸引力 | 吸引强度百分比 |
| 吸引时长 | 吸引持续时间（秒） |
| 环绕 | 粒子到达边界后折返 |

### 生命周期

- 存活时长范围（秒）
- 淡入时长 — 粒子出现时的渐变过渡
- 淡出时长 — 粒子消失时的渐变过渡

### 高级效果

- **闪烁** — 亮度周期性波动，可调幅度和速度
- **模糊** — Mask Feather 实现，仅圆形/多边形可用，支持模糊比例控制
- **随机种子** — 改变种子值可整体重置所有随机分布
- **发射偏移** — 粒子出现时间的随机偏移（秒），用于制造不齐效果

### 预设系统

**快捷预设（4 个内建）**：

| 预设 | 粒子数 | 特点 |
|------|--------|------|
| 经典星空 | 300 | 蓝色系，向上飘动 |
| 彩色星云 | 500 | 全色相，360°扩散 |
| 极光飘动 | 400 | 绿色系，横向运动 |
| 金色粒子雨 | 250 | 金色，高速下落 |

**槽位预设（4 个用户槽位）**：保存/加载当前参数，数据持久化存储在 AE 设置中。

---

## 控制器层 `Ctrl_Starfield`

生成粒子时自动创建一个 Null 层，包含 28 个参数滑块：

- 粒子参数滑块（数量、尺寸、缩放等）
- 颜色 HSL + 色相扩散
- 运动控制（方向、速度、吸引）
- 生命周期（时长、淡入淡出）
- 高级效果（闪烁、模糊、种子）

> 所有粒子动画由 AE 表达式驱动，调整滑块后**即时生效**，无需重新生成。

---

## 技巧与提示

- 大量粒子（>500）建议使用圆形以优化性能
- 用预设快速定位效果方向，再细调参数
- 目标吸引搭配遮罩范围发射可做出"粒子汇聚成形状"的效果
- `Ctrl_Starfield` 层的表达式可直接编辑，实现更复杂的自定义行为
