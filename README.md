# 星空粒子生成器 v3.1.2

**Starry Sky Particle Generator** — Adobe After Effects 2026 ScriptUI Panel

在合成中快速生成大量可控的星空粒子，支持多种形状、颜色、运动模式和特效。

## 安装

将 `starry-sky-generator.jsx` 复制到：

```
C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\
```

然后重启 AE，在 **窗口 > 星空粒子生成器** 中打开。

## 快速开始

1. 打开或新建一个合成
2. 调整面板中的参数（数量、尺寸、颜色等）
3. 点击 **▶ 生成粒子**
4. 合成中会自动创建 `Ctrl_Starfield` 控制器层，所有参数可在其中微调

## 功能一览

### 粒子参数
| 控制项 | 说明 |
|--------|------|
| 数量 | 粒子总数（10~2000） |
| 尺寸 | 最小 ~ 最大尺寸（像素） |
| 缩放 | 出生大小 → 消亡大小的缩放范围 |
| 随机缩放 | 每个粒子缩放的随机变异范围 |
| 形状 | 方形 / 圆形 / 五边形 / 六边形 |
| 颜色 | HSL 调色板 + 色相扩散（每个粒子颜色不同） |

### 发射区域
- **全合成**：粒子均匀分布于画面
- **遮罩范围**：从指定图层的遮罩内发射

### 运动控制
- 主运动方向 + 方向扩散（随机角度）
- 速度范围（像素/秒）
- **目标吸引**：粒子可被目标图层的遮罩吸引

### 生命周期
- 存活时长范围（秒）
- 淡入 / 淡出时长

### 高级效果
- **闪烁**：亮度周期性波动（幅度 + 速度）
- **模糊**：蒙版羽化模糊（仅圆形/多边形），支持模糊比例控制
- 随机种子（重置所有随机分布）

### 预设
- 4 个内建预设：经典星空 / 彩色星云 / 极光飘动 / 金色粒子雨
- 4 个用户槽位：保存/加载当前参数

## 技巧

- 生成后通过 `Ctrl_Starfield` 层可全局调整所有粒子参数
- 粒子的运动、颜色、透明度等由表达式驱动，调整后即时生效
- 大量粒子（>500）建议使用圆形以优化性能
