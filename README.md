# 星空粒子生成器 (Starry Sky Generator)

适用于 **Adobe After Effects 2026**（中文版/英文版）的 ExtendScript 面板插件，在合成中生成可控的星空粒子效果。

## 安装

1. 将 `starry-sky-generator.jsx` 复制到：
   ```
   C:\Program Files\Adobe\Adobe After Effects 2026\Support Files\Scripts\ScriptUI Panels\
   ```
2. 重启 After Effects
3. 菜单：**窗口 > 星空粒子生成器**

## 功能一览

### 粒子控制
- 粒子数量（10~2000，默认 50）
- 尺寸范围（最小/最大 px）
- 缩放动画：初始大小% → 最终大小%（生命周期内插值）
- 粒子形状：圆形 / 正方形 / 五边形 / 六边形（通过 Mask 实现）
- 色相 / 色相随机范围 / 饱和度 / 亮度 + RGB 颜色选取器

### 运动控制
- 主运动方向（0~360°）+ 方向随机扩散（扇形角度）
- 速度范围（最小/最大 px/s）
- 发射区域：**全合成** / **遮罩范围**（精确射线法 Point-in-Polygon 判定）
- 发射密度（遮罩模式下有效，控制粒子填充率）
- 发射随机偏移（每粒子独立时间偏移，错开初始出生时刻）
- 屏幕边界环绕开关

### 目标吸引
- 三种模式：无 / Null 点 / 遮罩范围
- 吸引力强度（0~100%，相对漂移速度的百分比）
- 吸引时长（吸引力持续秒数，过后恢复自由漂移）

### 生命周期
- 每粒子独立随机生命周期（最小~最大秒数）
- 淡入/淡出时长
- Perlin 噪声驱动的闪烁效果（幅度/速度可调）

### 预设系统
- **4 个内置预设**：经典星空 / 彩色星云 / 极光飘动 / 金色粒子雨
- **4 个槽位预设**：存储/使用当前参数，保存在 `app.settings` 中跨会话持久化
- 槽位选择器对话框（保存预设/加载预设按钮弹出）

### 其他
- 自动创建合成（无活动合成时自动创建 1920×1080, 30fps, 8s）
- 清除全部（一键删除所有粒子层 + 控制器）
- 完整可复制的错误报告（含 AE 环境 + 运行日志）
- 控制器层 `Ctrl_Starfield`（绿色标签 Null），所有参数滑块实时可调

## 快速使用

1. 打开 AE，打开或创建一个合成
2. 打开 **窗口 > 星空粒子生成器**
3. 调好粒子参数，点击 **▸ 生成粒子**
4. 播放预览，拖拽控制器滑块的参数实时观察变化
5. 点预设按钮一键切换风格，或存到槽位供以后使用
